## Eric's Time Logging Application

## Setup
+ Clone this repository
+ In the root directory: install all of the dependencies using ```bundle install```

## To run:
+ In the root directory:
```bundle exec ruby run_logger.rb
```

+ In the root directory:
```bundle exec rspec```
